# To-Do List 应用

这是一个使用原生 HTML、CSS 和 JavaScript 构建的简单而功能完整的待办事项应用。

## 功能特性

- ✅ 添加新的待办事项
- ✅ 标记待办事项为已完成/未完成
- ✅ 删除待办事项
- ✅ 双击待办事项进行编辑
- ✅ 筛选功能：查看全部、未完成或已完成的待办事项
- ✅ 清除所有已完成的待办事项
- ✅ 待办事项数量统计
- ✅ 本地存储：刷新页面后数据不会丢失
- ✅ 响应式设计：适配不同屏幕尺寸
- ✅ 防止 XSS 攻击的安全措施

## 如何使用

1. 打开 `index.html` 文件即可使用应用
2. 在输入框中输入待办事项，点击「添加」按钮或按 Enter 键添加
3. 点击复选框标记待办事项的完成状态
4. 双击待办事项文本进行编辑
5. 点击「删除」按钮删除待办事项
6. 使用筛选按钮切换显示不同状态的待办事项
7. 点击「清除已完成」按钮删除所有已完成的待办事项

## 技术栈

- **HTML5**: 用于构建页面结构
- **CSS3**: 用于样式设计和响应式布局
- **JavaScript**: 用于实现交互功能
- **LocalStorage**: 用于本地数据存储
